Grid

O que é grid layout

É um modelo de layout para web responsiva.

display: grid;

Propriedade usada em um contâiner para torná-lo grid, onde os elementos dele podem ser rearranjados. Por padrão dispõe todos os elementos em bloco (block).

Propriedades de disposição de elementos de um Gridbox:

  • justify-content: [valor] ;

    Redistribui todo o espaçamento lateral para um grid container (cria um fluxo horizontal).

    Aplicada no grid-container

    Valores:

    • start: Deixa todas as colunas de elementos no início do flex container.
    • end: Deixa todas as colunas de elementos no final.
    • center: Deixa todos os elementos no meio.
    • space-around: Deixa todas as colunas de elementos com um espaçamento fixo (margem) para todas elas, em ambos os lados.
    • space-evenly: Deixa todas as colunas de elementos com um espaçamento igual entre si.
    • space-between: Deixa duas das colunas de elementos no ínicio e final, e coloca as remanescentes com espaçamento igual entre si.

    justify-self: [valor];

    Altera a posição apenas de um item.

    Aplicada no grid-item

  • align-items [valor] ;

    Redistribui os elementos em relação à um eixo vertical em cada grid item.

    Aplicada no grid-container

    Valores:

    • start: Dispõe todos os elementos na parte superior do seu respectivo container de grid-item.
    • end: Dispõe todos os elementos na parte inferior do seu respectivo container de grid-item.
    • center: Dispõe todos os elementos na parte central do seu respectivo container de grid-item.

    align-self: [valor];

    Altera a posição apenas de um item.

    Aplicada no grid-item

  • grid-template-columns: [valor] [valor]... ;

    Cria mais colunas para o grid-container.

    Aplicada no grid-container

    Valores:

    • 100px 100px 100px: Cria 3 colunas com 100px de largura cada. O grid-container terá largura do elemento com o display: grid, mas suas colunas terão 300px no total.
    • auto 100px: Cria 2 colunas, uma com o tamanho do elemento (auto), e outra com 100px de largura.

  • grid-template-rows: [valor] [valor]... ;

    Cria mais linhas para o grid-container.

    Aplicada no grid-container

    Valores:

    • 1fr 1fr 1fr: Cria 3 linhas com 1/3 da altura do grid container cada.

    Nova unidade de medida: fr (Fração). Serve também para colunas.

  • grid-column: [grid-column-start] / [grid-column-end] ;

    Determina em que coluna o grid-item começa e termina. Útil para ocupar mais de uma coluna por item.

    grid-row-start é um valor incluso

    grid-row-end é um valor excluso

    Aplicada no grid-item

    Valores:

    • span 2: Substitui os dois valores da propriedade
    • 1 / 3: O grid-item que começa na coluna 1, termina de ser exibido na coluna 2 (Ocupa duas colunas portanto).

  • grid-row: [grid-row-start] / [grid-row-end] ;

    Determina em que linha o grid-item começa e termina. Útil para ocupar mais de uma linha por item.

    grid-row-start é um valor incluso

    grid-row-end é um valor excluso

    Aplicada no grid-item

    Valores:

    • span 2: Substitui os dois valores da propriedade
    • 1 / 3: O grid-item que começa na linha 1, termina de ser exibido na linha 2 (Ocupa duas linhas portanto).

  • gap: [valor] ;

    Aplica um espaçamento interno entre todos os grid-items. Não aplica espaçamento nas bordas do grid-container.

    Aplicada no grid-container

    Valores:

    • 10px: Aplica um espaçamento de 10px entre os elementos, para linhas e colunas.

    column-gap: [valor] ;

    Aplica um espaçamento interno entre todas as colunas dos grid-items. Não aplica espaçamento nas bordas do grid-container.

    Aplicada no grid-container

    row-gap: [valor] ;

    Aplica um espaçamento interno entre todas as linhas dos grid-items. Não aplica espaçamento nas bordas do grid-container.

    Aplicada no grid-container
