Animações

Propriedades CSS:

  • @keyframes [nome-da-animação] {

    keyframes-selectors {
        css-styles;
    }

}
    keyframes-selectors css-styles
    from {} transform: função();
    to {} top; width; background; left; etc
    0%

    *Obs: Pode ser usado apenas o "to {} ou 100% {} "

  • animation: [nome do keyframe] [duração] [iteration];
    • animation-name: [Nome do keyframe];
    • animation-duration:1s;
    • animation-timing-function:
      function Descrição
      linear Curvas de velocidade para a animação
      ease
      ease-in
      ease-out
      ease-in-out
      step-start
      step-end
      cubic-bezier(x1,y1,x2,y2)
    • animation-delay: 1s;
    • animation-iteration-count: number / infinite;
    • animation-direction: reverse / alternate / alternate-reverse;
    • animation-fill-mode:
      Valor Descrição
      forwards Os estilos que permanecerão ao terminar a animação, serão os do final (ultimo keyframe).
      backwards Os estilos que permanecerão ao terminar a animação, serão os do início (primeiro keyframe).
      both Antes de começar a animação permenecerão os estilos do início, e após terminar os do final.
    • animation-play-state: running / paused;
