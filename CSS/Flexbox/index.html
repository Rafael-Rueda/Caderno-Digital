Flexbox

O que é flexbox

É um modelo de layout para a Web responsiva.

Display: flex;

Propriedade usada em um contêiner para transformá-lo em flex, onde os elementos dentro dele podem ser rearranjados. Por padrão dispõe todos os elementos em linha (inline).

Propriedades de disposição de elementos de um flexbox:

  • justify-content: [valor] ;

    Redistribui todo o espaçamento lateral de um flex container entre todos os seus elementos (cria um fluxo horizontal).

    Aplicada no flex-container

    Valores:

    • flex-start: Deixa todos os elementos no início do flex container.
    • flex-end: Deixa todos os elementos no final.
    • center: Deixa todos os elementos no meio.
    • space-around: Deixa cada um dos elementos com um espaçamento (margem) fixo para todos, em ambos os lados (esquerda e direita).
    • space-evenly: Deixa cada um dos elementos com um espaçamento igual entre si, e entre as bordas do flex container.
    • space-between: Deixa os elementos ocuparem os espaços das extremidades direita e esquerda, centralizando depois os elementos no meio com espaçamento igual.

    justify-self: [valor] ;

    Aplicada no flex-item

    Valores:

    Mesmos valores.

  • align-items: [valor] ;

    Ajusta os elementos a partir de um eixo transversal horizontal e central do flex container (cria um alinhamento vertical).

    Aplicada no flex-container

    Valores:

    • flex-start: Deixa os elementos no topo do flex container.
    • flex-end: Deixa os elementos em baixo.
    • center: Alinha os elementos no centro da linha transversal no flex container.
    • stretch: Faz com que os elementos ocupem a altura máxima do flex container (estica-os).
    • baseline: Alinha os elementos sobrepondo a baseline de texto dos elementos na linha transversal

    align-self: [valor] ;

    Aplicada no flex-item

    Valores:

    Mesmos valores.

  • flex-wrap: [valor] ;

    Quebra a linha quando a disposição dos elementos atinge o limite do flex container, dispondo os remanescentes elementos para uma próxima linha.

    Aplicada no flex-container

    Valores:

    • nowrap: (default) Os itens não irão para uma próxima linha no overflow.
    • wrap: Os itens irão para uma próxima linha no overflow.
    • wrap-reverse: Os itens irão para uma nova linha, mas está virá antes da linha atual, ou seja, "retrocedem" uma linha quando necessária a quebra.

  • flex-direction: [valor] ;

    Muda a direção da disposição de elementos do flex container.

    Aplicada no flex-container

    Valores:

    • row: (default) Os itens são dispostos em linha.
    • column: Os itens são dispostos em coluna (inverte os eixos transversais).
    • row-reverse: Os itens são dispostos em linha, mas começam da direita para a esquerda.
    • column-reverse: Os itens são dispostos em coluna, mas começam de baixo pra cima.
