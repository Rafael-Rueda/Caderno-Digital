Structured Query Language

Comandos:

  • CREATE

    CREATE DATABASE [NOME]

    Cria um banco de dados

  • (CREATE DATABASE) ON

    CREATE DATABASE [NOME] ON (
NAME = 'string',
FILENAME = '\Diretorio',
SIZE = '10MB',
MAXSIZE = '50MB',
FILEGROWTH = '5MB'
);

    Cria um banco de dados

  • SELECT

    SELECT * FROM [NOME DA TABELA]

    Seleciona e imprime as colunas especificadas juntamente com seus registros. Aceita filtros WHERE.

  • TABLE

    CREATE TABLE [NOME] (
[Nome da coluna] data-type,
[Nome da coluna] data-type,
...
);

    Cria uma Tabela com as colunas predefinidas

  • INSERT

    INSERT INTO [NOME DA TABELA] ([NomeColuna1], [NomeColuna2], ...) 
VALUES
('registro_coluna1','registro_coluna2', ...),
('registro2_coluna1','registro2_coluna2', ...)

    Insere registros nos campos de uma tabela

  • UPDATE

    UPDATE [NOME DA TABELA] SET [NOME DA COLUNA] = 'valor'

    Atualiza uma tabela, podendo definir novamente o valor dos registros. Aceita filtros WHERE.

  • DELETE

    DELETE FROM [NOME DA TABELA] WHERE [Nome da coluna (id)] = 'valor'

    Apaga um registro de uma tabela. Aceita filtros WHERE.

  • ALTER

    ALTER TABLE [NOME DA TABELA] ALTER COLUMN [NOME DA COLUNA] data-type NOT NULL

    Altera uma tabela, podendo alterar tambem o data-type da coluna.

  • DROP

    ALTER TABLE [NOME DA TABELA] DROP COLUMN [NOME DA COLUNA]

DROP TABLE [NOME DA TABELA]

    Exclui uma tabela ou coluna

  • CONSTRAINT: PRIMARY KEY

    ALTER TABLE [NOME DA TABELA] CONSTRAINT [PK_NOME_DA_COLUNA] PRIMARY KEY CLUSTERED ([NOME DA COLUNA])

    Define uma chave restrição do tipo chave-primária para uma coluna não nula de uma tabela.
