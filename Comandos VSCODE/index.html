Comandos e atalhos do VSCODE:

Atalho Resultado Extensão
CTRL + Space Seleção rápida de diretório Não
CTRL + Shift + P Abre o console de comandos. Não
[Comando] Wrap with Abbreviation Cria uma tag HTML para as palavras selecionadas EMMET
[Comando] Show Live Server Preview Permite a visualização do site no VSCODE Live Server Preview
CTRL + : Torna a linha automaticamente em comentário. Não
CTRL + D Seleciona a próxima ocorrência de uma palavra selecionada. Se o cursor estiver em uma palavra, também a seleciona. Não
ALT + Downarrow Troca a posição da linha com a de baixo. Não
CTRL + ALT + Downarrow Adiciona mais um cursor na linha de baixo. Não

EMMET: Atalhos para criar código HTML:

>

article.c_img>img[src="img.png" alt="Imagem"]+.container01>p{Texto qualquer}
(Tab)

Resultado


<article class="c_img">
    <img src="img.png" alt="Imagem">
    <div class="container01">
        <p>Texto qualquer</p>
    </div>
</article>

Atalhos:

Caractere Função
"string" Nome da tag HTML
> Elemento filho
[ ] Atributos
+ Elemento irmão
{ } Conteúdo do elemento
. Classe
# ID
( ) Agrupa conteúdo
* Multiplica elementos
$ Como valor, numera itens.
