<source/>

Elemento vazio que define uma relação entre o documento e um arquivo externo. (Apenas usado nas mídias de áudio, vídeo e imagens dinâmicas).

Propriedades:

Propriedade Valor Descrição
media (CSS properties) Especifica as propriedades referentes à janela de visualização, com prorpiedades css. (max-width: 100px)
Operadores (and / , / not Permitem especificar mais propriedades referentes à janela de visualização.
srcset URL Define o caminho para uma imagem.
src URL Define o caminho para um vídeo ou áudio.
type media_type Define o media type do arquivo a ser relacionado. (Valores media_type: https://www.iana.org/assignments/media-types/media-types.xhtml)
<audio>

Elemento vazio que define uma relação entre o documento e um arquivo externo.

Propriedades:

Propriedade Valor Descrição
autoplay Inicia a reprodução automaticamente. (Em alguns navegadores só acontece quando a mídia também está mutada).
controls Permite a visualização de controles de reprodução padrão do navegador.
loop Quando termina a reprodução da mídia, ela recomeça automaticamente.
muted Muta a saída de áudio.
preload metadata Carrega apenas alguns dados como nome e duração da mídia antes da página carregar.
auto Carrega todo o arquivo automaticamente antes da página carregar.
none Não carrega o arquivo até que se inicie a reprodução.
src URL Define o caminho para o arquivo.
