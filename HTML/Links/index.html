Links são caminhos para ir de um lugar a outro na web

<a>

Cria uma âncora de link

Propriedades:

''
Propriedade Valor Descrição
href URL Define o caminho para onde o link irá levar.
# Define um caminho vazio.
rel external Define que o link leva a um conteúdo externo.
next Próxima parte do site.
prev Parte anterior do site.
search Define que o link é uma ferramenta de pesquisa para o site.
nofollow Search spiders não seguem o link
download "String" Define se é um link de download, e pelo valor, dá o nome do arquivo a baixar pelo usuário.
type media_type Define o media type do arquivo a ser baixado. (Valores media_type: https://www.iana.org/assignments/media-types/media-types.xhtml)
target _blank Link aberto em uma nova janela.
_self Link aberto na mesma janela.
<link>

Elemento vazio que define uma relação entre o documento e um arquivo externo.

Propriedades:

Propriedade Valor Descrição
href URL Define o caminho para onde o link irá levar.
# Define um caminho vazio.
rel stylesheet Define que a relação é com um arquivo CSS
icon Define que a relação é com um arquivo de ícone da página.
type media_type Define o media type do arquivo a ser relacionado. (Valores media_type: https://www.iana.org/assignments/media-types/media-types.xhtml)
