Tags para formulários

<form>

Tag usada para criar formulários.

Propriedades:

Atributo Valor Descrição
action URL Caminho para o qual as informações do formulário são enviadas.
name "string" Especifica o nome do formulário.
target _self Mesma página
_blank Nova página
novalidate
autocomplete on Com autocomplete que o navegador dá ao usuário.
off Sem autocomplete.
rel external Define que o link leva a um conteúdo externo.
next Próxima parte do site.
prev Parte anterior do site.
search Define que o link é uma ferramenta de pesquisa para o site.
nofollow Search spiders não seguem o link
<input>

Cria um campo onde o usuário pode preencher informações.

Propriedades:

Atributo Valor Descrição
type button Botão
checkbox Checkbox
color Cor
date Data
datetime-local Data e horário
email Email
file Arquivo
hidden Oculto
image Botão feito de imagem
month Mês e ano
number Número
password Senha
radio Opção única
range Intervalo numérico
reset Botão para redefinição de valores
search Texto usado para pesquisa
submit Enviar formulário
tel Telefone
text Texto (padrão)
time Horário
url URL
week Semana
value * (string ou number) Valor que o input recebe, e transmitirá quando o formulário for enviado.
checked Pre-seleciona o valor como true para o input em que esse atributo está (checklists e radios).
src URL Define a imagem do botão. (Apenas para o type="image").
alt string Texto alternativo para inputs com type="image".
form form_id Especifica qual o formulário que o input pertence.
autofocus Quando a página carrega, o input com esse valor ganha foco automaticamente.
accept media_type Define o media type aceito para o input de type="file". https://www.iana.org/assignments/media-types/media-types.xhtml)
file_extension Exemplo: .png ; .jpeg ; .mp4
image/* Caractere " * " especifica que todos os media types estão inclusos (mas limitados à arquivos do tipo imagem nesse caso).
min/max number Valor mínimo/máximo para inputs que suportam (type="range" por exemplo).
minlength number Quantidade mínima/máxima de caracteres para o input.
required Informa que o campo do input é obrigatório.
step number Especifica um valor que será a variação de um intervalo numérico,. limitando os valores possíveis (ir de 2 em 2, 3 em 3, etc).
placeholder string Especifica um texto explicativo do input, que só é visível quando o input está vazio.
name string Especifica o nome de um input. Apenas inputs com um nome terão os seus valores enviados ao servidor quando o formulário é preenchido.
pattern regexp Define a estrutura dos dados a serem inseridos (por exemplo, quando uma senha deve conter pelo menos 1 caractere especial).
<label>

Tag que define uma legenda para vários elementos.

Elementos que podem ser associados a um label:

  • input
  • meter
  • progress
  • select
  • textarea

Propriedades:

Atributo Valor Descrição
for id do elemento Associa um label a um elemento por meio do id.
form id do formulário Indica a que formulário o label pertence.
<select>

Cria um menu de opções para seleção única (drop-down list). Não usa a tag input, mas pode ser utilizado para recolher informações do usuário em um formulário.

Parâmetros:

Atributo Valor Descrição
name string Especifica o nome do select, usado para conter seu valor no par "nome/valor" no servidor.
disabled Desativa a drop-down list.
form id do formulário Declara a que formulário a drop-down list pertence.
size number Define o número de opções visíveis no select.
multiple Multiplas opções podem ser selecionadas.
autofocus Quando a página carrega, o select com esse valor ganha foco automaticamente.
<option>

Opção para o elemento "select"

Parâmetros:

Atributo Valor Descrição
label string Especifica uma curta descrição para a opção (nome da opção).
disabled Desativa a opção.
selected Declara uma opção pre-selecionada.
value string Define o valor para encaminhar ao servidor.
<fieldset>

Cria um agrupamentos para vários inputs dentro de um formulário.

<legend>

Dá um título/legenda para um fieldset. Pode ser declarado logo após o fieldset.
