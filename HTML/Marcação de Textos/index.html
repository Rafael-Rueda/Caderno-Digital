Tags de marcação de texto:

<abbr>

Tag usada para marcar a abreviação de um termo, fornecendo sua definição por meio de um parâmetro.

Exemplo: A <abbr title = "Hyper-Text Markup Language">HTML</abbr> é uma linguagem de marcação.

Display: Inline

Parâmetros:

Atributo Valor Descrição
title "string" Define o significado da abreviação.

<dfn>

Definição de um termo em sua primeira ocorrência.

Exemplo: A <dfn>HTML</dfn> é uma linguagem de marcação.

Display: Inline

Parâmetros:

  • Sem parâmetros específicos

<bdi>

Isola termos que são impressos na direção contrária ao resto dos textos, mantendo-os em uma única direção.

Exemplo: Trechos de escrita árabe <bdi>إيان</bdi>, que são lidos da direita para a esquerda, no meio de textos cuja escrita é ocidental, podem ser isolados, mantendo a integridade da direção do texto como um todo.

Display: Inline

Parâmetros:

  • Sem parâmetros específicos

<bdo>

Detemina uma direção para a impressão do texto, por meio de um parâmetro.

Exemplo: <bdo dir="rtl">O texto é impresso da direita para a esquerda</bdo>

Display: Inline

Parâmetros:

Atributo Valor Descrição
dir rtl Muda a direção da impressão do texto.
ltf

<blockquote>

Define longas citações. Deve ser informado de onde vem a citação por meio de um parâmetro.

Display: Block

Parâmetros:

Atributo Valor Descrição
cite "URL" Informa a origem/autor da citação.

<q>

Define curtas citações.

Display: Inline

Parâmetros:

Atributo Valor Descrição

<code>

Define um texto como a expressão de um código que o computador lê.

Exemplo: A propriedade CSS <code>background-color</code> é usada para definir uma cor de fundo.
