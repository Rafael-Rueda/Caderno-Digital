Objeto: Node (MDN)

Node é uma interface da qual diversos tipos do DOM herdam, e que permite que esses tipos sejam tratados de forma similar, por exemplo, herdando os mesmos métodos ou sendo testados da mesma forma.

Propriedades:

  • childNodes : Retorna um Objeto Nodelist
  • firstChild : Retorna o primeiro nó filho direto do nó.
  • lastChild
  • nextSibling
  • previousSibling
  • parentNode
  • parentElement
  • nodeName : Representa o nome do nó. É uma propriedade somente leitura, o que significa que não pode ser definida ou modificada.
  • nodeValue : Define / Retorna o valor do nó atual
  • nodeType: Retorna um número que representa o tipo do nó
    • Número Representação
    1 Nó de elemento
    2 Nó de atributo
    3 Nó de texto

Métodos:

  • appendChild(child) : Adiciona ao elemento pai, um nó filho específico, como seu último filho.
  • removeChild(child) : Remove um elemento filho do documento
  • cloneNode(deep) : Clona um nó e todo seu conteúdo.
    deep = "true" - Clona o nó com todo o seu conteúdo;
    deep = "false" - Clona apenas o nó.
  • contains(descendente) : Retorna true ou false, toda vez que o nó indicado nos parâmetros for descendente de outro.
  • hasChildNodes()
  • insertBefore(novo, referencia) : Insere um novo nó no elemento pai antes de outro nó na árvore DOM, referenciado nos parâmetros.
  • isEqualNode(el) : Retorna true or false, toda vez que os dados e o type de um nó coincidirem com o de outro.

Objeto: Nodelist (MDN)

Propriedades:

  • length

Métodos:

  • item(index) : Retorna um item de uma Nodelist pelo índice
  • forEach() : Executa uma função uma vez para cada elemento da Nodelist.

Objeto: Element (MDN)

O objeto Element é um tipo de objeto Node (portanto herda as suas propriedades e métodos) que representa um elemento HTML no DOM. Ele fornece propriedades e métodos especificamente projetados para trabalhar com elementos HTML.

Propriedades:

  • attributes : Retorna um Objeto NamedNodeMap contendo os atributos do elemento.
  • childElementCount : Retorna o número de elementos filho de um elemento.
  • children : Retorna o número de nós filhos de um elemento.
  • classList: Retorna um Objeto DOMTokenList contendo uma lista com todas as classes do elemento.
  • className : Retorna o valor do atributo "class" de um elemento (String).
  • id : retorna o valor do atributo "id" de um elemento.
  • clientHeight : Retorna a altura de um elemento.
  • clientWidth
  • firstElementChild
  • lastElementChild
  • nextElementSibling
  • previousElementSibling
  • localname : Representa o nome do elemento.
  • scrollHeight : Retorna a altura do scroll de um elemento.
  • scrollWidth
  • scrollLeft : Representa o eixo (posição) de scroll horizontal do elemento.
  • scrollTop
  • innerHTML : Retorna / Altera a marcação HTML de um elemento.
  • tagName : Retorna o nome de Tag de um elemento.

Métodos:

  • append(par1, par2, ...) : Adiciona múltiplos nós na página, como filhos de um elemento pai.
  • addEventListener('type', listener) : Para objetos que são EventTarget
    type = https://developer.mozilla.org/en-US/docs/Web/Events
  • removeEventListener('type', listener)
  • after(node1, node2, ...) : Insere um elemento logo ao final de outro.
  • before(node1, node2, ...)
  • closest() : Retorna o ancestral mais próximo do elemento.
  • getAttribute('attr_name') : Retorna o valor de um atributo
  • getAttributeNames() : Retorna um Objeto Array com todos os nomes dos atributos de um elemento.
  • hasAttribute('attr_name') : Retorna True ou False, checando se um elemento tem certo atributo ou não.
  • hasAttributes() : Verifica se o elemento tem algum atributo. Retorna True ou False.
  • setAttribute('attr_name', value)
  • remove() : Remove um elemento
  • removeAttribute('attr_name')
  • replaceChildren(par1, par2, ...) : Substitui todos os nós filhos de um elemento, e coloca novos no lugar.
  • scroll(x, y): Define coordenadas de scroll.
  • scrollBy(x, y) : Adiciona uma quantidade de scroll.
  • scrollIntoView() : Define um scroll em um elemento de modo que ele fique na tela do usuário.

Objeto: Array (MDN)

Propriedades:

  • length
  • prototype : Permite a adição de propriedades e métodos como parte do Objeto Array

Métodos:

  • isArray() : Retorna True ou False
  • at(index) : Retorna o valor de um item do Array pelo índice. Aceita valores negativos (contagem a partir do final, começando por -1).
  • concat(array2) : Junta dois arrays
  • every(função) : Checa se todos os itens do array satisfazem uma função, e retorna true, caso todos eles a satisfamçam.
  • some(função) : Checa se pelo menos um itenm do array satisfaz uma função, e retorna true, caso ele a satisfamça.
  • fill(valor, posição_inicial, posição_final) : Preenche (substitui) os itens de um array com um valor.
  • filter(função) : Cria um novo array apenas com os elementos que satisfazem uma função.
  • find(função) : Retorna o valor do primeiro elemento no array que satisfaz uma função.
  • findIndex(função) : Retorna o indice do primeiro elemento no array que satisfaz uma função.
  • findLast()
  • findLastIndex()
  • forEach()
  • includes(valor) : Retorna true ou false.
  • indexOf(valor, ocorrencia) : Retorna o index da primeira (ou desejada) ocorrência de um valor no array.
  • lastIndexOf(valor)
  • push()
  • reverse() : Inverte a ordem dos elementos do array.
  • slice(posição_inicial, posição_final) : Copia uma parte de um array e cria um novo com essa parte.
  • splice(index, quantidade) : Remove um elemento do array.
  • sort() : Altera a ordem dos elementos de um array aleatoriamente.
