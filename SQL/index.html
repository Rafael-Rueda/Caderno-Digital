Structured Query Language

Comandos:

  • CREATE

    CREATE DATABASE [NOME]

  • (CREATE DATABASE) ON

    CREATE DATABASE [NOME] ON (
NAME = 'string',
FILENAME = '\Diretorio',
SIZE = '10MB',
MAXSIZE = '50MB',
FILEGROWTH = '5MB'
);

  • SELECT

    SELECT * FROM [NOME DA TABELA]

  • TABLE

    CREATE TABLE [NOME] (
[Nome da coluna] data-type,
[Nome da coluna] data-type,
...
);

  • INSERT

    INSERT INTO [NOME DA TABELA] ([NomeColuna1], [NomeColuna2], ...) 
VALUES
('registro_coluna1','registro_coluna2', ...),
('registro2_coluna1','registro2_coluna2', ...)

  • UPDATE

    UPDATE [NOME DA TABELA] SET [NOME DA COLUNA] = 'valor'

  • DELETE

    DELETE FROM [NOME DA TABELA] WHERE [Nome da coluna (id)] = 'valor'

  • ALTER

    ALTER TABLE [NOME DA TABELA] ALTER COLUMN [NOME DA COLUNA] data-type NOT NULL

  • CONSTRAINT: PRIMARY KEY

    ALTER TABLE [NOME DA TABELA] CONSTRAINT [PK_NOME_DA_COLUNA] PRIMARY KEY CLUSTERED ([NOME DA COLUNA])
